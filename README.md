# Unemployment Analysis with Python

This project focuses on analyzing unemployment data in India using Python. It explores two datasets: one containing unemployment data up to November 2020 and another with data spanning 2019 to 2020. The analysis includes data preprocessing, visualization, and insights generation.

## Overview

Unemployment is a critical economic and social indicator. This project aims to provide insights into unemployment rates across different states and regions in India. The analysis covers:

- Data loading and cleaning
- Data exploration
- Visualization of unemployment trends
- Regional and state-wise analysis

## Getting Started

These instructions will help you get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Python (3.x recommended)
- Jupyter Notebook
- Libraries: pandas, numpy, matplotlib, seaborn, plotly

You can install the required libraries using pip:

```bash
pip install pandas numpy matplotlib seaborn plotly
```

### Installation

1. Clone the repository:

```bash
git clone https://github.com/karan7864/oibsip_taskno_2.git
```

2. Open the Jupyter Notebook:

```bash
jupyter notebook Unemployment_Analysis.ipynb
```

3. Run the notebook cell by cell to perform the analysis step by step.

- Explore the provided Jupyter Notebook for a detailed walkthrough of the analysis process.
